这个仓库用来放一些歌谱源文件和虚拟歌手工程文件<br>
## MuseScore
[MuseScore](https://github.com/musescore/MuseScore) 是一个开源的打谱软件，`museScore`文件夹内就是该软件的项目文件，有一些扒谱和一些记谱

## OpenUtau
[OpenUtau](https://github.com/openutau/OpenUtau) 是一个开源的歌声合成软件，因此`OpenUtau`文件夹内存的是 [OpenUtau](https://github.com/openutau/OpenUtau) 的工程文件<br>
加~前缀的内容不完整，或有部分错误内容未修正
>通常是先由自动扒谱软件（如 [Vocal2Midi](https://github.com/Xiantaidu/Vocal2Midi) ）生成，而后续的人工校对只完成了一部分<br>

有一些文件有`[带音高曲线]`后缀，这不意味着其他文件无音高曲线，而是扒原曲的音高曲线
>还有一些无此后缀的也带有原曲的音高曲线<br>

`self`文件夹内是原创曲或原创填词
>除非你是从视频简介来的，不然还是不要看这个文件夹了
## SparkNMN
`SparkNMN`文件夹内是简谱编辑器 [SparkNMN](https://github.com/yezhiyi9670/sparks-notation-1) 的项目源文件<br>
在此文件夹中的歌谱通常也发布在 [简谱之家](https://www.jianpu.net/space/1526)<br>
在 [简谱之家](https://www.jianpu.net/space/1526) 发布的简谱都将源码存储在这里
