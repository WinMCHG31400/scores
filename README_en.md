This repository is created for save some source files for score and singing synthesis.
## MuseScore
[MuseScore](https://github.com/musescore/MuseScore) is an open-source music composition and notation software, so there are some project file for this software in `museScore`folder.

## OpenUtau
[OpenUtau](https://github.com/openutau/OpenUtau) is an open-sourse singing synthesis software, so there are some project files for this software in `OpenUtau` folder<br>
the files with `~` prefix are incomplete or have some errors that have not been fixed yet
>usually, the files are generated by automatic score extraction software (like [Vocal2Midi](https://github.com/Xiantaidu/Vocal2Midi)), and the subsequent manual proofreading has only been partially completed<br>

some files have the suffix `[带音高曲线]`, which does not mean that other files do not have pitch curves<br>
>some files without this suffix also have the original song's pitch curve<br>

`self` folder contains original songs or original lyrics<br>
>Unless you came from the video description, you probably shouldn't look at this folder

## SparkNMN
`SparkNMN` folder contains the project source files for the numeric score editor [SparkNMN](https://github.com/yezhiyi9670/sparks-notation-1)<br>
files in this folder are usually also published on [简谱之家](https://www.jianpu.net/space/1526)<br>
source code of the numeric scores published on [简谱之家](https://www.jianpu.net/space/1526) are stored here

## Format
the file format of [OpenUtau](https://github.com/openutau/OpenUtau) and [SparkNMN](https://github.com/yezhiyi9670/sparks-notation-1) is text-based, but [MuseScore](https://github.com/musescore/MuseScore) saves files in a compressed package format
